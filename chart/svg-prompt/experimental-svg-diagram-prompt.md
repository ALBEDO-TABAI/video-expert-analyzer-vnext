## SVG 形式实验影像感知干预结构图 — AGENT PROMPT

### 任务描述

根据你完成的形式实验影像分析报告，生成一张
**形式约束（操作源）/ 感知干预（效果机制）/ 认识论效应（观者终点）**
三栏对照的 SVG 结构图。

这张图的底层逻辑是 Shklovsky（1917）的**感知延长链**：
每一行代表影像的一个感知段落，三列分别回答——
- 左列：这一段部署了什么**形式约束**？（固定机位？循环？媒介暴露？算法规则？）
- 中列：这个约束对观者的感知习惯施加了什么**具体干预**？处于干预弧线的哪个阶段？
- 右列：干预最终将观者推向了什么**认识论位置**？（意识到自身感知？质疑媒介本体？进入新感知状态？）

> **三栏锚定说明**：
> 左列锚定 MODULE 1（形式约束识别）+ MODULE 2（媒介物质性审计）；
> 中列锚定 MODULE 4（感知干预地图——核心模块）；
> 右列锚定 MODULE 5（观者位置分析）+ MODULE 7（边界测试）。

> **箭头方向论证**：
> 形式实验影像的核心操作是"约束→干预→效应"的**单向因果流**——
> 形式约束决定感知干预的方式，感知干预决定观者最终的认识论位置。
> 这不是 MV 式的"画面与语言互相注释"（双向），
> 而是实验影像特有的"形式决定感知、感知改写认知"的单向递进。
> 因此水平箭头严格为 左→中→右（单向），不可反向。

---

### 图表规格

- 格式：SVG，viewBox="0 0 680 [动态高度]"
- 宽度：width="100%"，不得写死像素值
- 无外层容器 div，直接输出 `<svg>` 元素
- 背景：透明（不设任何背景色）
- 必须包含 `role="img"`、`<title>` 和 `<desc>` 无障碍标签

---

### 结构规则

**整体布局：三栏 + 标题行**

```
[标题行]  形式约束（操作源） |  感知干预（效果机制） |  认识论效应（观者终点）
[分隔线]
[段落 1 行]  左卡片  →  中卡片  →  右卡片
[段落 2 行]  左卡片  →  中卡片  →  右卡片
...
[段落 N 行]  左卡片  →  中卡片  →  右卡片
```

**坐标系**
- 安全区：x=40 到 x=640，y=40 到 y=(viewBox高度-20)
- 三列 x 起点：左列 x=40，中列 x=256，右列 x=472
- 每列卡片宽度：168px
- 列间连接箭头区域：x=208→254（左到中），x=424→470（中到右）

**行间距计算**
- 标题行高度：约 35px（含分隔线）
- 每段落行高度：56px（卡片高）+ 22px（行间距）= 78px
- 第 N 段落的卡片 y 起点 = 50 + (N-1) × 90
  （根据实际段落数调整，确保最后一行底边 + 20px = viewBox 高度）

### 卡片规格

每个卡片为双行文字卡片：

```svg
<g class="node c-{COLOR}">
  <rect x="{X}" y="{Y}" width="168" height="56" rx="8" stroke-width="0.5"/>
  <text class="th" x="{X+84}" y="{Y+19}"
        text-anchor="middle" dominant-baseline="central">
    主标题（≤10字）
  </text>
  <text class="ts" x="{X+84}" y="{Y+39}"
        text-anchor="middle" dominant-baseline="central">
    副标题（≤16字）
  </text>
</g>
```

卡片点击绑定 sendPrompt，内容为"详细分析 [主标题] 段落"：
```svg
<g class="node c-{COLOR}" onclick="sendPrompt('详细分析 [主标题] 段落')">
```

### 颜色分配规则

每一个感知段落分配一个颜色，同一行的三列卡片使用**同一颜色**。
颜色编码**感知干预的阶段与强度**，而非视觉装饰。按以下逻辑选色：

| 感知干预阶段特征 | 推荐色 | 逻辑依据 |
|---|---|---|
| 初始约束建立/感知惯性尚存 | c-coral | 暖色 = 观者仍处于舒适区，约束刚刚开始生效 |
| 感知习惯被扰动/不适浮现 | c-pink | 不安的温度上升，自动化感知开始松动 |
| 持续干预/感知适应与再陌生化 | c-purple | 高密度干预区，感知框架本身被质疑 |
| 媒介物质性暴露/去幻觉主义顶点 | c-gray | 冷色克制 = 幻觉被剥除后的"裸露"状态 |
| 感知位移完成/新感知状态涌现 | c-teal | 冷静的认识论转变，观者抵达新位置 |
| 边界测试/类别质疑/开放性收束 | c-green | "这还是电影吗" = 类别边界的绿色生长地带 |
| 可根据实际干预弧线灵活调整 | — | — |

禁止对不同段落使用相同颜色（颜色必须区分各感知段落）。

### 连接箭头规格

**水平箭头**（同行，左→中→右，表示"约束→干预→效应"因果方向）：

```svg
<line x1="208" y1="{卡片中心Y}" x2="254" y2="{卡片中心Y}"
      class="arr" marker-end="url(#arrow)"
      stroke="var(--color-border-secondary)"/>
<line x1="424" y1="{卡片中心Y}" x2="470" y2="{卡片中心Y}"
      class="arr" marker-end="url(#arrow)"
      stroke="var(--color-border-secondary)"/>
```

> **设计意图**：水平箭头在 MV 图中是"对照关系"（画面 ↔ 主题 ↔ 语言），
> 但在形式实验图中是**因果链**——形式约束产生感知干预，感知干预产生认识论效应。
> 这是 Shklovsky 陌生化理论的操作化：形式手法（priem）→ 延长感知 → 恢复对事物的感觉。
> 因此箭头方向严格为 左→中→右（单向），不可反向。

**垂直箭头**（同列，上段→下段，三列各一条）：

```svg
<line x1="{列中心X}" y1="{上段卡片底边+2}" x2="{列中心X}" y2="{下段卡片顶边-2}"
      class="arr" marker-end="url(#arrow)"
      stroke="var(--color-border-secondary)"/>
```

三列中心 X 值：左列 124，中列 340，右列 556。

> **设计意图**：垂直箭头表示感知体验在时间中的累积推进。
> 左列的垂直箭头 = 形式约束的叠加/切换/递进；
> 中列的垂直箭头 = 感知干预的累积/适应/再陌生化循环；
> 右列的垂直箭头 = 观者认识论位置的逐步位移。

### 内容填充规则

根据你的形式实验影像分析报告，将作品的时间轴划分为 3–6 个感知段落
（不是机械等分时间，而是按**感知体验的质变节点**来切分——
即观者的感知状态发生可辨认变化的那些时刻）。
为每个段落填写三张卡片：

**左卡片（形式约束 — 操作源）**
- 主标题：该段落的主导形式约束名称（≤8字）
  例如："固定机位/持续" "循环×50次" "JPEG逐代压缩" "闪烁12Hz" "算法迭代"
- 副标题：该约束排除/暴露了什么（≤14字）
  例如："排除剪辑·暴露真实时长" "排除清晰度·暴露压缩逻辑"

**中卡片（感知干预 — 效果机制）**
- 主标题：该段落对观者感知施加的干预类型（≤8字）
  例如："视觉节律替代" "时间感拉伸" "注意力漂移" "感知适应/再陌生化"
- 副标题：时间戳范围，格式 "MM:SS – MM:SS"

**右卡片（认识论效应 — 观者终点）**
- 主标题：该段落将观者推向的认识论位置（≤10字）
  例如："意识到在看" "媒介不再透明" "类别边界模糊" "感知框架重置"
- 副标题：对应的理论锚点或观者位置类型（≤14字）
  例如："Shklovsky·延长感知过程" "Gidal·去幻觉主义" "Gunning·吸引力模式"

> **填充原则**：
> 1. 左列写"部署了什么形式规则"——锚定 MODULE 1 的约束清单 + MODULE 2 的物质性前景化
> 2. 中列写"这条规则对感知做了什么"——锚定 MODULE 4 的感知干预地图
> 3. 右列写"观者因此抵达了什么位置"——锚定 MODULE 5 的观者位置 + MODULE 7 的边界测试
> 4. 如果某段落的感知干预尚未累积到产生认识论位移（如开头几段仅在建立约束条件），
>    右列卡片的主标题写"（蓄能中）"，副标题写"惯性感知尚未被打断"
> 5. 对于极简作品（如整部作品仅一个镜头），段落切分需在微观时间尺度上寻找
>    感知体验的变化——如"前5分钟：期待内容" → "5-15分钟：意识到无内容" →
>    "15-30分钟：注意力转向自身" → "30-45分钟：进入/抵抗冥想状态"

### 段落切分指引

按以下优先级寻找切分点（优先级从高到低）：

1. **感知质变时刻**：观者的感知模式发生可辨认切换的节点
   （如从"看内容"切换到"意识到在看"，从"视觉主导"切换到"听觉主导"）
2. **形式约束切换/叠加点**：一条新约束被引入，或现有约束发生变化
   （如从固定机位突然切到运动镜头，从彩色突然切到黑白）
3. **媒介物质性暴露点**：媒介的物质构成从背景突然变成前景
   （如画面开始出现明显的颗粒/压缩伪影/帧率异常）
4. **感知适应临界点**：某种干预因重复而失去陌生化效力，
   作品被迫切换到新的干预方式的那一刻
5. **边界测试时刻**：作品开始质疑自身类别归属
   （如影像完全消失只剩声音，或声音消失只剩纯色画面）

典型的形式实验影像段落划分：

| 子类型 | 典型段落数 | 典型结构 |
|--------|-----------|---------|
| 结构电影（短片 10-30min） | 3–4 段 | 约束建立 → 感知适应 → 再陌生化 → 结构完成 |
| 持续时间型（长片 1hr+） | 4–5 段 | 期待惯性 → 惯性消解 → 注意力漂移 → 感知转向 → 时间体验变形 |
| 闪烁电影（短片 5-15min） | 3 段 | 频率建立 → 生理反应峰值 → 适应/余震 |
| 过程型/衰减型 | 4–5 段 | 清晰起点 → 衰减开始 → 物质性暴露 → 抽象化顶点 → 新秩序涌现 |
| 算法/生成式 | 3–5 段 | 规则可辨期 → 规则超出认知 → 涌现/失控 → 人机边界质疑 |
| 发现素材/挪用 | 3–4 段 | 原始语境残留 → 语境剥离 → 新意义涌现 → 互文共振 |

### viewBox 高度计算公式

```
段落数 = N
viewBox 高度 = 50 + N × 90 + 20
```

示例：4 段 → 50 + 4×90 + 20 = 430
示例：5 段 → 50 + 5×90 + 20 = 520
示例：3 段 → 50 + 3×90 + 20 = 340

### 输出禁忌

- 禁止在 SVG 内部写解释性文字段落
- 禁止使用 `<style>` 块定义颜色（使用 `c-{color}` class）
- 禁止硬编码十六进制颜色值
- 禁止在任何 `<text>` 元素上省略 class 属性
- 禁止让卡片文字超出 168px 宽度（主标题≤10字，副标题≤16字）
- 禁止将两个不同段落设为同一颜色
- 垂直箭头必须三列都画，不可只画中列
- 禁止将水平箭头画为双向——因果链方向严格为 左→中→右
- 禁止在右列"认识论效应"卡片中写画面描述——右列只写观者的感知/认识论位置变化
- 禁止在左列"形式约束"卡片中写主观感受——左列只写客观的形式规则与操作
- 禁止将形式实验影像的段落等同于叙事"幕"——切分依据是感知体验的质变，不是叙事转折
