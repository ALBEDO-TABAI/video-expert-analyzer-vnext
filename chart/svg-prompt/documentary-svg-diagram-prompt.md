## SVG 纪录片可信度建构结构图 — AGENT PROMPT

### 任务描述

根据你完成的纪录片/纪实/访谈分析报告，生成一张
**证据素材（可信度来源）/ 论证弧线（修辞功能）/ 可信度着陆（认识论效果）**
三栏对照的 SVG 结构图。

这张图的底层逻辑是**可信度建构管线**（基于 Plantinga, 1997 与 Nichols, 2017）：
每一行代表纪录片的一个论证段落，三列分别回答——
- 左列：这一段部署了什么类型的**证据素材**？从什么来源借用了可信度？
- 中列：这一段执行的**论证功能**是什么？处于论证弧线的哪个阶段？
- 右列：这些证据和论证最终在观众心中产生了什么**认识论效果**？

> **三栏锚定说明**：
> 左列锚定 MODULE 6（证据系统与可信度装置）；
> 中列锚定 MODULE 3（论证结构分析）；
> 右列锚定 MODULE 2（声音层级系统）+ MODULE 4（访谈框架分析）的综合产出。

> **箭头方向论证**：水平箭头严格为 左→中→右（单向），
> 因为纪录片的核心修辞操作是一条**因果链**——
> 证据素材（来源）→ 被论证结构编排（机制）→ 在观众心中产生可信度效果（终点）。
> 这是一条单向的"可信度生产管线"，而非对等的对照关系，
> 因此不适用 MV 基底模板的双向箭头。

---

### 图表规格

- 格式：SVG，viewBox="0 0 680 [动态高度]"
- 宽度：width="100%"，不得写死像素值
- 无外层容器 div，直接输出 `<svg>` 元素
- 背景：透明（不设任何背景色）
- 必须包含 `role="img"`、`<title>` 和 `<desc>` 无障碍标签

---

### 结构规则

**整体布局：三栏 + 标题行**

```
[标题行]  证据素材（可信度来源） |  论证弧线（修辞功能） |  可信度着陆（认识论效果）
[分隔线]
[段落 1 行]  左卡片  →  中卡片  →  右卡片
[段落 2 行]  左卡片  →  中卡片  →  右卡片
...
[段落 N 行]  左卡片  →  中卡片  →  右卡片
```

**坐标系**
- 安全区：x=40 到 x=640，y=40 到 y=(viewBox高度-20)
- 三列 x 起点：左列 x=40，中列 x=256，右列 x=472
- 每列卡片宽度：168px
- 列间连接箭头区域：x=208→254（左到中），x=424→470（中到右）

**行间距计算**
- 标题行高度：约 35px（含分隔线）
- 每段落行高度：56px（卡片高）+ 22px（行间距）= 78px
- 第 N 段落的卡片 y 起点 = 50 + (N-1) × 90
  （根据实际段落数调整，确保最后一行底边 + 20px = viewBox 高度）

### 卡片规格

每个卡片为双行文字卡片：

```svg
<g class="node c-{COLOR}">
  <rect x="{X}" y="{Y}" width="168" height="56" rx="8" stroke-width="0.5"/>
  <text class="th" x="{X+84}" y="{Y+19}"
        text-anchor="middle" dominant-baseline="central">
    主标题（≤10字）
  </text>
  <text class="ts" x="{X+84}" y="{Y+39}"
        text-anchor="middle" dominant-baseline="central">
    副标题（≤16字）
  </text>
</g>
```

卡片点击绑定 sendPrompt，内容为"详细分析 [主标题] 段落"：
```svg
<g class="node c-{COLOR}" onclick="sendPrompt('详细分析 [主标题] 段落')">
```

### 颜色分配规则

每一个论证段落分配一个颜色，同一行的三列卡片使用**同一颜色**。
颜色编码**论证阶段的修辞功能**，而非视觉装饰。按以下逻辑选色：

| 论证阶段特征 | 推荐色 | 逻辑依据 |
|---|---|---|
| 框架设定/问题引入/开场钩子 | c-coral | 高能量开场：用震撼画面或反直觉信息建立观看框架 |
| 背景铺陈/事实呈现/证据部署 | c-teal | 冷静的证据充能阶段：数据、档案、专家陈述密集展开 |
| 情感升级/见证者证词/pathos 卷入 | c-pink | 情绪温度上升：个体经验的呈现制造共情 |
| 论证高潮/关键揭示/冲突顶点 | c-purple | 高密度修辞集中点：核心发现或矛盾的爆发时刻 |
| 论证收束/结论着陆/立场表达 | c-green | 可信度着陆完成：观众被引导至导演期望的认知位置 |
| 开放收尾/反思悬置/行动号召 | c-gray | 克制退出：从论证叙事退出到留白或行动指令 |
| 可根据实际论证弧线灵活调整 | — | — |

禁止对不同段落使用相同颜色（颜色必须区分各论证段落）。

### 连接箭头规格

**水平箭头**（同行，左→中→右，表示可信度建构方向）：

```svg
<line x1="208" y1="{卡片中心Y}" x2="254" y2="{卡片中心Y}"
      class="arr" marker-end="url(#arrow)"
      stroke="var(--color-border-secondary)"/>
<line x1="424" y1="{卡片中心Y}" x2="470" y2="{卡片中心Y}"
      class="arr" marker-end="url(#arrow)"
      stroke="var(--color-border-secondary)"/>
```

> **设计意图**：水平箭头在 MV 图中是"对照关系"（画面 ↔ 主题 ↔ 语言），
> 但在纪录片图中是**因果流向**——证据素材被论证结构编排，
> 再产出可信度效果。方向严格为 左→中→右（单向），不可反向。
> 这与广告 SVG 的"意义转移管线"异曲同工，但流转的不是"品牌意义"而是"认识论可信度"。

**垂直箭头**（同列，上段→下段，三列各一条）：

```svg
<line x1="{列中心X}" y1="{上段卡片底边+2}" x2="{列中心X}" y2="{下段卡片顶边-2}"
      class="arr" marker-end="url(#arrow)"
      stroke="var(--color-border-secondary)"/>
```

三列中心 X 值：左列 124，中列 340，右列 556。

> **设计意图**：垂直箭头表示论证进程的推进。
> 左列的垂直箭头 = 证据类型的切换与叠加（从事实到证词到档案到数据……）；
> 中列的垂直箭头 = 论证阶段的递进（从问题提出到证据呈现到结论着陆）；
> 右列的垂直箭头 = 可信度的逐层累积（从初始信任建立到深度认同）。

### 内容填充规则

根据你的纪录片分析报告，将纪录片的时间轴划分为 3–6 个论证段落
（不是机械等分时间，而是按**论证功能的转换**来切分）。
为每个段落填写三张卡片：

**左卡片（证据素材 — 可信度来源）**
- 主标题：该段落部署的核心证据类型（≤8字）
  例如："专家访谈" "档案影像" "现场实拍" "数据图表" "见证者证词" "文件特写"
- 副标题：该证据的认识论属性（≤14字）
  例如："学科权威·logos 诉求" "第一人称经验·pathos" "历史证物·事实锚定"

**中卡片（论证弧线 — 修辞功能）**
- 主标题：该段落在论证弧线中的功能角色（≤8字）
  例如："框架设定" "证据部署" "情感卷入" "矛盾揭示" "论证收束" "开放悬置"
- 副标题：时间戳范围，格式 "MM:SS – MM:SS"

**右卡片（可信度着陆 — 认识论效果）**
- 主标题：该段落在观众心中产生的可信度效果（≤10字）
  例如："建立权威信任" "激发共情认同" "制造认知冲击" "锚定因果归因" "巩固核心论点"
- 副标题：对应 Nichols 模式或 Aristotle 诉求路径（≤14字）
  例如："解说式·ethos 权威通道" "参与式·pathos 见证通道" "观察式·在场性信任"

> **填充原则**：
> 1. 左列写"部署了什么证据、从哪里借用可信度"——锚定 MODULE 6 的证据清单
> 2. 中列写"这一段在论证弧线中执行什么功能"——锚定 MODULE 3 的论证段落划分
> 3. 右列写"观众因此在认识论层面发生了什么变化"——锚定 MODULE 2 + MODULE 4 的综合判断
> 4. 如果某段落尚未完成可信度建构（如前段仅在铺陈背景，核心论点尚未亮出），
>    右列卡片的主标题写"（蓄能中）"，副标题写"信任基础搭建中"

### 段落切分指引

按以下优先级寻找切分点（优先级从高到低）：

1. **论证转折点**：核心证据首次出现、关键矛盾被揭示的时刻，必须是切分点
2. **声音层级切换**：旁白→访谈、访谈→环境音、音乐介入/消失等声音权力转移
3. **证据类型切换**：从专家话语切到见证者证词，或从档案影像切到实地拍摄
4. **Renov 倾向切换**：段落的主导倾向从"记录"变为"说服"，或从"分析"变为"表达"
5. **再现模式切换**：从解说式切到观察式、从参与式切到反身式
6. **情绪/色调断裂点**：音乐风格突变、画面色温骤变、剪辑节奏陡变

典型的纪录片段落划分：

| 纪录片类型 | 典型段落数 | 典型结构 |
|----------|-----------|---------|
| 调查型纪录片（45–90min） | 5–6 段 | 问题引入 → 背景铺陈 → 证据展开 → 矛盾揭示 → 深度追问 → 结论/开放 |
| 人物纪录片（30–60min） | 4–5 段 | 人物登场 → 生平回溯 → 核心事件 → 情感高潮 → 现状/展望 |
| 观察式纪录片（60–90min） | 3–4 段 | 日常切入 → 事件浮现 → 张力累积 → 自然收束 |
| 访谈节目（20–60min） | 3–5 段 | 话题导入 → 经历叙述 → 核心观点 → 反思/总结 |
| 短纪录片/纪实短视频（5–15min） | 3 段 | 钩子/问题 → 证据/叙述 → 结论/收束 |

### viewBox 高度计算公式

```
段落数 = N
viewBox 高度 = 50 + N × 90 + 20
```

示例：5 段 → 50 + 5×90 + 20 = 520
示例：3 段 → 50 + 3×90 + 20 = 340
示例：6 段 → 50 + 6×90 + 20 = 610

### CALIBRATION EXAMPLE

以下是一个 4 段调查型纪录片的填充示例（仅供校准参考，勿照搬）：

```
段落 1（c-coral）
  左：[实地拍摄]「废弃厂区·环境音同步」
  中：[框架设定]「00:00 – 04:30」
  右：[建立在场性信任]「观察式·在场性通道」

段落 2（c-teal）
  左：[专家访谈×3]「环境学者·logos 权威佐证」
  中：[证据部署]「04:30 – 18:20」
  右：[锚定因果归因]「解说式·ethos 权威通道」

段落 3（c-pink）
  左：[居民证词]「第一人称受害经历·pathos」
  中：[情感卷入]「18:20 – 31:00」
  右：[激发共情认同]「参与式·pathos 见证通道」

段落 4（c-green）
  左：[文件特写+数据]「政府文件·事实锚定」
  中：[论证收束]「31:00 – 42:15」
  右：[巩固核心论点]「logos 收束·行动暗示」
```

> 推导说明：该纪录片的可信度管线为——
> 先用观察式实地画面建立"我带你来到现场"的在场性信任（ethos 铺底），
> 再用专家群体的 logos 佐证搭建因果框架，
> 然后用居民个体的 pathos 证词完成情感层面的信任充能，
> 最后用文件和数据的 logos 收束锁定论点——
> 整条管线呈现"ethos → logos → pathos → logos"的修辞诉求交替节奏。

### 输出禁忌

- 禁止在 SVG 内部写解释性文字段落
- 禁止使用 `<style>` 块定义颜色（使用 `c-{color}` class）
- 禁止硬编码十六进制颜色值
- 禁止在任何 `<text>` 元素上省略 class 属性
- 禁止让卡片文字超出 168px 宽度（主标题≤10字，副标题≤16字）
- 禁止将两个不同段落设为同一颜色
- 垂直箭头必须三列都画，不可只画中列
- 禁止将水平箭头画为双向——可信度建构方向严格为 左→中→右
- 禁止在右列"可信度着陆"卡片中写画面描述——右列只写认识论效果
- 禁止将左列写成"旁白内容"——左列写的是**证据类型与来源属性**，不是文字转写
