## SVG 节庆宣传广告动员结构图 — AGENT PROMPT

### 任务描述

根据你完成的节庆宣传广告分析报告，生成一张
**日常锚点（分离触发）/ 动员弧线（仪式功能）/ 节庆承诺（体验着陆）**
三栏对照的 SVG 结构图。

这张图的底层逻辑是 Turner（1969）的**仪式过程三阶段**：
每一行代表广告的一个动员段落，三列分别回答——
- 左列：这一段从什么**日常状态**中把观众拉出来？用了什么分离触发？
- 中列：这一段执行的**动员功能**是什么？处于仪式动员弧线的哪个阶段？
- 右列：这一段为观众**承诺**了什么节庆体验？观众在这一段结束后离"我要参加"更近了多少？

> **三栏锚定声明**：
> - 左列锚定 MODULE 1（节奏语法与视觉动员）+ MODULE 4（多轨交织分析）
> - 中列锚定 MODULE 2（仪式动员分析）
> - 右列锚定 MODULE 3（节庆世界的空间建构）+ MODULE 6（怀旧与时间修辞）

### 图表规格

- 格式：SVG，viewBox="0 0 680 [动态高度]"
- 宽度：width="100%"，不得写死像素值
- 无外层容器 div，直接输出 `<svg>` 元素
- 背景：透明（不设任何背景色）
- 必须包含 `role="img"`、`<title>` 和 `<desc>` 无障碍标签

### 结构规则

**整体布局：三栏 + 标题行**

```
[标题行]  日常锚点（分离触发） |  动员弧线（仪式功能） |  节庆承诺（体验着陆）
[分隔线]
[段落 1 行]  左卡片  →  中卡片  →  右卡片
[段落 2 行]  左卡片  →  中卡片  →  右卡片
...
[段落 N 行]  左卡片  →  中卡片  →  右卡片
```

**坐标系**
- 安全区：x=40 到 x=640，y=40 到 y=(viewBox高度-20)
- 三列 x 起点：左列 x=40，中列 x=256，右列 x=472
- 每列卡片宽度：168px
- 列间连接箭头区域：x=208→254（左到中），x=424→470（中到右）

**行间距计算**
- 标题行高度：约 35px（含分隔线）
- 每段落行高度：56px（卡片高）+ 22px（行间距）= 78px
- 第 N 段落的卡片 y 起点 = 50 + (N-1) × 90
  （根据实际段落数调整，确保最后一行底边 + 20px = viewBox 高度）


### 颜色分配规则

每一个动员段落分配一个颜色，同一行的三列卡片使用**同一颜色**。
颜色编码**动员阶段的仪式功能**，而非视觉装饰。按以下逻辑选色：

| 动员阶段特征 | 推荐色 | 逻辑依据 |
|---|---|---|
| 日常中断/注意力捕获/感官刺点 | c-coral | 高能量开场，打破日常信息流的第一击 |
| 氛围升温/阈限入口/世界预览 | c-teal | 从日常温度冷却进入仪式准备的过渡色调 |
| 欲望卷入/交融体承诺/社交想象 | c-pink | 情感温度上升，集体归属感的暖色共振 |
| 仪式高潮/情绪引爆/全轨共振 | c-purple | 所有感官轨道在此汇聚的高密度顶点 |
| 节庆承诺结晶/体验着陆/身份转换完成 | c-green | 从"旁观者"到"参与者"的转化完成 |
| CTA/信息落地/行动触发/品牌锁定 | c-gray | 从情绪高潮退出到具体行动指令的克制收尾 |
| 可根据实际动员弧线灵活调整 | — | — |

禁止对不同段落使用相同颜色（颜色必须区分各动员段落）。

### 连接箭头规格

**水平箭头**（同行，左→中→右，表示仪式转化方向）：

```svg
<line x1="208" y1="{卡片中心Y}" x2="254" y2="{卡片中心Y}"
      class="arr" marker-end="url(#arrow)"
      stroke="var(--color-border-secondary)"/>
<line x1="424" y1="{卡片中心Y}" x2="470" y2="{卡片中心Y}"
      class="arr" marker-end="url(#arrow)"
      stroke="var(--color-border-secondary)"/>
```

> **设计意图与箭头方向论证**：
> 水平箭头严格为**单向 左→中→右**，不可反向或双向。
> 理由：节庆广告的核心操作是一条单向的仪式转化管线——
> 意义从"日常状态的分离触发"出发，经由"动员机制的仪式功能"加工，
> 最终着陆为"节庆体验的承诺"。
> 这一方向性直接映射 Turner 仪式过程的不可逆三阶段：
> 分离（separation）→ 阈限（liminality）→ 聚合（incorporation）。
> 观众不会从"节庆承诺"回溯到"日常状态"——
> 转化一旦发生，日常就被暂时悬置了。

**垂直箭头**（同列，上段→下段，三列各一条）：

```svg
<line x1="{列中心X}" y1="{上段卡片底边+2}" x2="{列中心X}" y2="{下段卡片顶边-2}"
      class="arr" marker-end="url(#arrow)"
      stroke="var(--color-border-secondary)"/>
```

三列中心 X 值：左列 124，中列 340，右列 556。

> **设计意图**：垂直箭头表示动员进程的推进。
> - 左列的垂直箭头 = 日常分离的深化（从表层注意力到深层情绪松动）
> - 中列的垂直箭头 = 动员阶段的递进（AIDA：注意→兴趣→欲望→行动）
> - 右列的垂直箭头 = 节庆承诺的累积/具体化（从模糊氛围到具体体验画面）

### 内容填充规则

根据你的节庆广告分析报告，将广告的时间轴划分为 3–6 个动员段落
（不是机械等分时间，而是按**动员功能的转换**来切分）。
为每个段落填写三张卡片：

**左卡片（日常锚点 — 分离触发）**
- 主标题：该段落用来制造"日常中断"的核心感官/视听手段（≤8字）
  例如："城市日常流" "手机碎片" "快速剪辑轰炸" "倒计时字幕"
- 副标题：该手段的动员功能一句话说明（≤14字）
  例如："用日常疲惫感制造逃逸欲望" "节奏加速打破浏览惯性"

**中卡片（动员弧线 — 仪式功能）**
- 主标题：该段落在动员弧线中的功能角色（≤8字）
  例如："日常中断" "阈限入口" "交融体预演" "仪式高潮" "行动触发"
- 副标题：时间戳范围，格式 "MM:SS – MM:SS"

**右卡片（节庆承诺 — 体验着陆）**
- 主标题：该段落为观众承诺的节庆体验（≤10字）
  例如："规模=不可错过" "集体=你不孤独" "感官=现场才有"
- 副标题：对应的承诺类型与修辞路径（≤14字）
  例如："communitas 承诺·归属感" "感官稀缺性·FOMO 驱动"

> **填充原则**：
> 1. 左列写"用什么打破了日常"——锚定 MODULE 1 的节奏语法 + MODULE 4 的多轨交织
> 2. 中列写"这一段在动员弧线中干什么"——锚定 MODULE 2 的仪式动员分析
> 3. 右列写"观众因此被承诺了什么"——锚定 MODULE 3 的空间建构 + MODULE 6 的怀旧修辞
> 4. 如果某段落尚未形成明确的节庆承诺（如开场仅在展示日常状态以制造分离感），
>    右列卡片的主标题写"（分离蓄能中）"，副标题写"节庆尚未显影"

### 段落切分指引

按以下优先级寻找切分点（优先级从高到低）：

1. **情绪引爆点**：所有感官轨道同时达到最高强度的那一刻，必须是一个切分点
2. **日常→节庆的世界切换**：画面从日常场景（办公室/通勤/家庭）切入节庆场景（舞台/人群/灯光）的那一帧
3. **音乐断裂/升级点**：音乐风格突变、节奏陡升、或环境音（欢呼/鞭炮/鼓声）突然介入
4. **CTA/信息落地时刻**：日期/地点/票务信息首次出现，标志从情绪动员转向行动触发
5. **时间修辞切换点**：从回溯（往届画面）切到当下，或从预告切到倒计时

典型的节庆广告段落划分：

| 广告类型 | 典型段落数 | 典型结构 |
|----------|-----------|---------|
| 15s 社交媒体预热片 | 3 段 | 感官刺点 → 核心画面 → CTA |
| 30s 节庆倒计时广告 | 4 段 | 日常中断 → 阈限入口 → 节庆预演 → 倒计时CTA |
| 60s 节庆氛围宣传片 | 4–5 段 | 日常→分离 → 氛围升温 → 交融体高潮 → 承诺结晶 → 行动号召 |
| 2–3min 城市节庆形象片 | 5–6 段 | 城市日常 → 传统回溯 → 筹备过渡 → 节庆展开 → 仪式高潮 → CTA+品牌锁定 |

### viewBox 高度计算公式

```
段落数 = N
viewBox 高度 = 50 + N × 90 + 20
```

示例：5 段 → 50 + 5×90 + 20 = 520
示例：3 段 → 50 + 3×90 + 20 = 340
示例：6 段 → 50 + 6×90 + 20 = 610

### 输出禁忌

- 禁止在 SVG 内部写解释性文字段落
- 禁止使用 `<style>` 块定义颜色（使用 `c-{color}` class）
- 禁止硬编码十六进制颜色值
- 禁止在任何 `<text>` 元素上省略 class 属性
- 禁止让卡片文字超出 168px 宽度（主标题≤10字，副标题≤16字）
- 禁止将两个不同段落设为同一颜色
- 垂直箭头必须三列都画，不可只画中列
- 禁止将水平箭头画为双向——仪式转化方向严格为 左→中→右（分离→动员→承诺）
- 禁止在右列"节庆承诺"卡片中写画面描述——右列只写观众被承诺了什么体验
- 禁止在左列"日常锚点"卡片中写节庆内容——左列只写被打破的日常状态或分离手段
