## SVG 现实纪录索引-伦理结构图 — AGENT PROMPT

### 任务描述

根据你完成的现实纪录分析报告，生成一张
**索引性证据（技术选择）/ 伦理位置（主体关系）/ 真实性效果（观众信念）**
三栏对照的 SVG 结构图。

这张图的底层逻辑是现实纪录的**真实性建构管线**：
每一行代表影片的一个观察段落（按拍摄者观察姿态的转变来划分），三列分别回答——
- 左列：这一段使用了什么**索引性技术选择**来锚定真实感？（长镜头？手持？自然光？同期声？）
- 中列：这一段的拍摄者处于什么**伦理位置**？与被拍摄者之间是什么关系？
- 右列：这些选择在观众心中建构了什么**真实性效果**？观众被引导相信什么？

> **三栏锚定声明**：
> 左列锚定 MODULE 1（索引性影像语法）+ MODULE 4（多轨证据分析）
> 中列锚定 MODULE 2（时间伦理）+ MODULE 3（主体间性）+ MODULE 6（拍摄伦理）
> 右列锚定 MODULE 5（真实性建构策略）+ MODULE 7（社会介入与见证功能）

### 图表规格

- 格式：SVG，viewBox="0 0 680 [动态高度]"
- 宽度：width="100%"，不得写死像素值
- 无外层容器 div，直接输出 `<svg>` 元素
- 背景：透明（不设任何背景色）
- 必须包含 `role="img"`、`<title>` 和 `<desc>` 无障碍标签

### 结构规则

**整体布局：三栏 + 标题行**

```
[标题行]  索引性证据（技术选择） |  伦理位置（主体关系） |  真实性效果（观众信念）
[分隔线]
[段落 1 行]  左卡片  →  中卡片  →  右卡片
[段落 2 行]  左卡片  →  中卡片  →  右卡片
...
[段落 N 行]  左卡片  →  中卡片  →  右卡片
```

**坐标系**
- 安全区：x=40 到 x=640，y=40 到 y=(viewBox高度-20)
- 三列 x 起点：左列 x=40，中列 x=256，右列 x=472
- 每列卡片宽度：168px
- 列间连接箭头区域：x=208→254（左到中），x=424→470（中到右）

**行间距计算**
- 标题行高度：约 35px（含分隔线）
- 每段落行高度：56px（卡片高）+ 22px（行间距）= 78px
- 第 N 段落的卡片 y 起点 = 50 + (N-1) × 90
  （根据实际段落数调整，确保最后一行底边 + 20px = viewBox 高度）


### 颜色分配规则

每一个观察段落分配一个颜色，同一行的三列卡片使用**同一颜色**。
颜色编码**拍摄者观察姿态的伦理阶段**，而非视觉装饰。按以下逻辑选色：

| 观察姿态特征 | 推荐色 | 逻辑依据 |
|---|---|---|
| 进入/建立在场/环境锚定 | c-coral | 摄影机初次进入被拍摄者的世界，能量状态=探索与接触 |
| 观察/等待/沉浸/死时间 | c-teal | 冷静的不干预阶段，摄影机作为"墙上的苍蝇"静观 |
| 靠近/跟随/身体共在 | c-pink | 物理距离缩短，拍摄者的身体介入程度上升 |
| 互动/对话/参与/挑衅 | c-purple | 模式从观察式滑向参与式，主体间性密度最高 |
| 见证/承受/脆弱时刻在场 | c-green | 伦理负重最大的段落——面对被拍摄者的痛苦/私密，摄影机选择留下 |
| 退出/反思/收束/社会指向 | c-gray | 从个体经验退回到社会语境，影片完成从"记录"到"见证"的转化 |
| 可根据实际观察姿态灵活调整 | — | — |

禁止对不同段落使用相同颜色（颜色必须区分各观察段落）。

### 连接箭头规格

**水平箭头**（同行，左→中→右，表示真实性建构的因果方向）：

```svg
<line x1="208" y1="{卡片中心Y}" x2="254" y2="{卡片中心Y}"
      class="arr" marker-end="url(#arrow)"
      stroke="var(--color-border-secondary)"/>
<line x1="424" y1="{卡片中心Y}" x2="470" y2="{卡片中心Y}"
      class="arr" marker-end="url(#arrow)"
      stroke="var(--color-border-secondary)"/>
```

> **设计意图与箭头方向论证**：
> 水平箭头严格为 **左→中→右（单向）**，不可反向或双向。
> 理由：现实纪录的真实性建构是一条**因果管线**——
> 技术选择（长镜头/手持/自然光）**产生**特定的伦理位置（不干预/共在/介入），
> 而伦理位置**决定**观众的真实性感知（"这是真的"/"这是被安排的"）。
> 因果方向不可逆转：观众的信念不会反向改变拍摄者当时的技术选择。
> 这与 MV 的双向对照关系（画面 ↔ 歌词互相注释）根本不同，
> 也与广告的意义转移管线（参照世界→修辞功能→品牌着陆）结构相似但语义不同。

**垂直箭头**（同列，上段→下段，三列各一条）：

```svg
<line x1="{列中心X}" y1="{上段卡片底边+2}" x2="{列中心X}" y2="{下段卡片顶边-2}"
      class="arr" marker-end="url(#arrow)"
      stroke="var(--color-border-secondary)"/>
```

三列中心 X 值：左列 124，中列 340，右列 556。

> **设计意图**：垂直箭头表示影片推进过程中三条线索的各自演变。
> 左列的垂直箭头 = 技术策略的切换/递进（如从固定机位过渡到手持跟随）；
> 中列的垂直箭头 = 伦理位置的漂移（如从观察式滑向参与式再退回见证式）；
> 右列的垂直箭头 = 观众信任的累积/断裂（真实感逐步建立或在某处出现裂缝）。

### 内容填充规则

根据你的现实纪录分析报告，将影片的时间轴划分为 3–6 个观察段落
（不是机械等分时间，而是按**拍摄者观察姿态的转变**来切分）。
为每个段落填写三张卡片：

**左卡片（索引性证据 — 技术选择）**
- 主标题：该段落的主导技术策略名称（≤8字）
  例如："固定机位长镜头" "手持贴身跟随" "自然光同期声" "远景静观"
- 副标题：该技术选择产出的索引性特征（≤14字）
  例如："3分12秒不间断·环境原光" "手持抖动·呼吸可闻"

**中卡片（伦理位置 — 主体关系）**
- 主标题：该段落拍摄者的伦理姿态（≤8字）
  例如："不干预观察" "对话共在" "脆弱见证" "介入协商" "退出反思"
- 副标题：时间戳范围 + Nichols 模式标记，格式 "MM:SS–MM:SS · 观察式"

**右卡片（真实性效果 — 观众信念）**
- 主标题：该段落在观众心中建构的核心信念（≤10字）
  例如："这不是排练的" "这个痛苦是真的" "日常≠无意义" "TA接纳了镜头"
- 副标题：该信念的建构机制一句话描述（≤14字）
  例如："偶然事件保留·技术瑕疵背书" "死时间拒绝戏剧化消费"

> **填充原则**：
> 1. 左列写"用了什么技术手段"——锚定 MODULE 1 的索引性影像语法 + MODULE 4 的多轨证据
> 2. 中列写"拍摄者此刻站在什么伦理位置"——锚定 MODULE 3 的主体间性 + MODULE 6 的伦理评估
> 3. 右列写"观众因此相信了什么"——锚定 MODULE 5 的真实性标记 + MODULE 7 的观众位置
> 4. 如果某段落的真实性效果被削弱（如出现后期音乐入侵、过度剪辑），
>    右列卡片的主标题写"（信任裂缝）"，副标题写具体的断裂原因

### 段落切分指引

按以下优先级寻找切分点（优先级从高到低）：

1. **Nichols 模式切换时刻**：观察式→参与式、参与式→反身式等模式转换的那一帧
2. **脆弱时刻边界**：被拍摄者进入/离开情感崩溃、身体痛苦、隐私暴露等脆弱状态
3. **拍摄者在场性变化**：摄影机从不可见变为可见、被拍摄者开始直视镜头/对拍摄者说话
4. **时间操作转换点**：从实时段落切入省略跳切、从叙事段落进入"死时间"
5. **音乐入侵点**：后期音乐首次出现或消失，标志从"记录"滑向"表达"

典型的现实纪录段落划分：

| 纪录类型 | 典型段落数 | 典型结构 |
|----------|-----------|---------|
| 机构观察型（Wiseman 式） | 4–5 段 | 进入机构 → 规则运行 → 裂缝显现 → 个体挣扎 → 机构继续 |
| 人物跟踪型 | 4–6 段 | 日常建立 → 事件萌发 → 靠近/跟随 → 高压/脆弱 → 变化/沉淀 → 退出 |
| 事件现场型 | 3–4 段 | 进入现场 → 事件展开·见证 → 余波/沉默 → 社会指向 |
| 日常生活型（慢纪录） | 3–5 段 | 进入·等待 → 日常循环 → 微事件 → 等待·沉默 → 时间本身 |

### viewBox 高度计算公式

```
段落数 = N
viewBox 高度 = 50 + N × 90 + 20
```

示例：5 段 → 50 + 5×90 + 20 = 520
示例：3 段 → 50 + 3×90 + 20 = 340
示例：6 段 → 50 + 6×90 + 20 = 610

### 输出禁忌

- 禁止在 SVG 内部写解释性文字段落
- 禁止使用 `<style>` 块定义颜色（使用 `c-{color}` class）
- 禁止硬编码十六进制颜色值
- 禁止在任何 `<text>` 元素上省略 class 属性
- 禁止让卡片文字超出 168px 宽度（主标题≤10字，副标题≤16字）
- 禁止将两个不同段落设为同一颜色
- 垂直箭头必须三列都画，不可只画中列
- **禁止将水平箭头画为双向**——真实性建构的因果方向严格为 左→中→右（技术选择→伦理位置→观众信念），不可逆转
- **禁止在右列写画面描述**——右列只写观众因此获得了什么信念/信任
- **禁止在中列省略 Nichols 模式标记**——每个段落的副标题必须标注当前的纪录模式（观察式/参与式/反身式/表演式）
