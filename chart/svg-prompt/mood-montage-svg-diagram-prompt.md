## SVG 情绪蒙太奇情感结构图 — AGENT PROMPT

### 任务描述

根据你完成的情绪蒙太奇分析报告，生成一张
**感觉质地（视听输入）/ 情感轨迹（弧线构造）/ 调谐效应（Stimmung）**
三栏对照的 SVG 结构图。

这张图的底层逻辑是**情感汇聚流**（Sinnerbrink, 2012; Plantinga, 2009）：
每一行代表蒙太奇的一个情感段落，三列分别回答——
- 左列：这一段部署了什么**感觉物质**？（色调/景别/运镜/质感/音效/物件）
- 中列：这些感觉物质**构造**了什么样的情感轨迹？处于情感弧线的哪个阶段？
- 右列：观众最终被**调谐**到什么样的情感状态（Stimmung）？

> **三栏锚定说明**
> - 左列锚定 MODULE 1（感觉物质统计）+ MODULE 2（情感地形学）+ MODULE 5（情感物件）
> - 中列锚定 MODULE 3（情感弧线识别）+ MODULE 6（节奏动力学）
> - 右列锚定 MODULE 3.4（Stimmung 模式判定）+ MODULE 4（多轨交织的合成效应）

> **箭头方向论证**
> 水平箭头严格为 左→中→右（单向）。理由：情绪蒙太奇的核心操作是
> "感觉物质汇聚 → 构造情感轨迹 → 产出调谐效应"的单向流，
> 不是 MV 中画面线与歌词线的平行互释关系。
> 感觉质地是**输入**，情感轨迹是**过程**，调谐效应是**产出**。

### 结构规则

**整体布局：三栏 + 标题行**

```
[标题行]  感觉质地（视听输入） |  情感轨迹（弧线构造） |  调谐效应（Stimmung）
[分隔线]
[段落 1 行]  左卡片  →  中卡片  →  右卡片
[段落 2 行]  左卡片  →  中卡片  →  右卡片
...
[段落 N 行]  左卡片  →  中卡片  →  右卡片
```

**坐标系**
- 安全区：x=40 到 x=640，y=40 到 y=(viewBox高度-20)
- 三列 x 起点：左列 x=40，中列 x=256，右列 x=472
- 每列卡片宽度：168px
- 列间连接箭头区域：x=208→254（左到中），x=424→470（中到右）

**行间距计算**
- 标题行高度：约 35px（含分隔线）
- 每段落行高度：56px（卡片高）+ 22px（行间距）= 78px
- 第 N 段落的卡片 y 起点 = 50 + (N-1) × 90
  （根据实际段落数调整，确保最后一行底边 + 20px = viewBox 高度）

### 颜色分配规则

每一个情感段落分配一个颜色，同一行的三列卡片使用**同一颜色**。
颜色编码**情感弧线的阶段特征**，而非视觉装饰。按以下逻辑选色：

| 情感阶段特征 | 推荐色 | 逻辑依据 |
|---|---|---|
| 入口/感觉苏醒/初始调谐 | c-coral | 暖色标记从日常注意力切入情感接收模式的那一刻 |
| 基调建立/浸泡/氛围铺展 | c-teal | 冷静的扩展色，对应情感的"基底温度"被设定的阶段 |
| 情感升温/张力积累/密度增长 | c-pink | 温度上升的粉色，对应情感强度递增的中段 |
| 情感峰值/泛音爆发/多轨共振最强点 | c-purple | 高密度紫色，对应 Eisenstein 泛音蒙太奇的最大合力点 |
| 情感转化/位移/质变 | c-green | 转化色，对应情感从一种状态变质为另一种状态 |
| 余韵/着陆/耗散/出口 | c-gray | 克制的灰色，对应情感释放后的余波或截断 |
| 可根据实际情感弧线灵活调整 | — | — |

禁止对不同段落使用相同颜色（颜色必须区分各情感段落）。

### 连接箭头规格

**水平箭头**（同行，左→中→右，表示感觉汇聚→情感构造→调谐产出）：

```svg
<line x1="208" y1="{卡片中心Y}" x2="254" y2="{卡片中心Y}"
      class="arr" marker-end="url(#arrow)"
      stroke="var(--color-border-secondary)"/>
<line x1="424" y1="{卡片中心Y}" x2="470" y2="{卡片中心Y}"
      class="arr" marker-end="url(#arrow)"
      stroke="var(--color-border-secondary)"/>
```

> **设计意图**：水平箭头严格为 左→中→右（单向），
> 表示"感觉物质 → 情感构造 → 调谐效应"的汇聚流方向。
> 不可画为双向——感觉质地是情感的原料，不是情感的产物。

**垂直箭头**（同列，上段→下段，三列各一条）：

```svg
<line x1="{列中心X}" y1="{上段卡片底边+2}" x2="{列中心X}" y2="{下段卡片顶边-2}"
      class="arr" marker-end="url(#arrow)"
      stroke="var(--color-border-secondary)"/>
```

三列中心 X 值：左列 124，中列 340，右列 556。

> **设计意图**：垂直箭头表示时间/情感进程的推进。
> - 左列的垂直箭头 = 感觉物质的切换/叠加/变质（色调转冷、质感跳变、新物件出现）
> - 中列的垂直箭头 = 情感弧线的推进（从低强度到高强度，或从一种情感状态过渡到另一种）
> - 右列的垂直箭头 = Stimmung 的演变/深化/翻转（从揭示性到自律性，从浸泡到溢出）

### 内容填充规则

根据你的情绪蒙太奇分析报告，将蒙太奇的时间轴划分为 3–6 个情感段落
（不是机械等分时间，而是按**情感状态的转换**来切分）。
为每个段落填写三张卡片：

**左卡片（感觉质地 — 视听输入）**
- 主标题：该段落的主导感觉特征（≤8字）
  例如："低饱和蓝灰" "碎片化闪切" "胶片颗粒感" "面部特写群" "环境音独占"
- 副标题：构成该感觉特征的核心视听元素（≤14字）
  例如："冷色调·手持微晃·雨声" "慢动作·去焦·弦乐渐入"

**中卡片（情感轨迹 — 弧线构造）**
- 主标题：该段落在情感弧线中的功能角色（≤8字）
  例如："感觉苏醒" "基调浸泡" "张力积累" "泛音爆发" "情感着陆" "余韵悬浮"
- 副标题：时间戳范围，格式 "MM:SS – MM:SS"

**右卡片（调谐效应 — Stimmung）**
- 主标题：该段落将观众调谐到的情感状态（≤10字）
  例如："怀旧性温柔" "不安的漂浮" "失控中的迷失" "存在性忧伤" "麻木的苏醒"
- 副标题：对应的 Stimmung 模式或多轨关系类型（≤14字）
  例如："揭示性·同步强化" "自律性·轨道对位" "过渡性·静默断裂"

> **填充原则**：
> 1. 左列写"部署了什么感觉原料"——锚定 MODULE 1 的视听统计 + MODULE 2 的空间情感温度
> 2. 中列写"这些原料构造了什么情感进程"——锚定 MODULE 3 的情感弧线 + MODULE 6 的节奏曲线
> 3. 右列写"观众最终被调谐到什么状态"——锚定 MODULE 3.4 的 Stimmung 判定 + MODULE 4 的多轨合成
> 4. 如果某段落处于情感建立初期、尚未形成明确的 Stimmung 状态，
>    右列卡片的主标题写"（调谐进行中）"，副标题写"感觉物质蓄能阶段"

### 段落切分指引

按以下优先级寻找切分点（优先级从高到低）：

1. **情感转折点**：情感弧线中效价或强度发生显著变化的时刻（MODULE 3.2 识别的转折点）
2. **色调/质感断裂点**：色调系统切换、画面质感跳变（如从高清到颗粒感、从彩色到黑白）
3. **节奏悬崖**：剪辑速率发生突然的大幅变化（从快切到长镜头，或反之）
4. **静默断裂点 / 音乐断裂点**：音乐或音效突然消失、突然切入、或风格骤变
5. **入口/出口标记**：蒙太奇的最初 3-8 秒和最后 5-15 秒通常各自构成独立段落

典型的情绪蒙太奇段落划分：

| 蒙太奇子类型 | 典型段落数 | 典型结构 |
|-------------|-----------|---------|
| 氛围型（30-60s） | 3 段 | 入口浸入 → 氛围驻留 → 余韵散出 |
| 碎片型（15-45s） | 3–4 段 | 感觉冲击 → 碎片堆积 → 泛音爆发 → 截断/着陆 |
| 记忆型（30-90s） | 4–5 段 | 现在锚定 → 记忆闪回 → 情感升温 → 时间叠合 → 余韵悬浮 |
| 生理型（15-60s） | 3–4 段 | 触觉唤醒 → 感觉密度攀升 → 生理共振峰值 → 渐弱/抽离 |
| 混合型（60-180s） | 5–6 段 | 入口 → 基调铺展 → 子类型 A 段 → 子类型 B 段 → 高潮合流 → 出口 |

### viewBox 高度计算公式

```
段落数 = N
viewBox 高度 = 50 + N × 90 + 20
```

示例：4 段 → 50 + 4×90 + 20 = 430
示例：5 段 → 50 + 5×90 + 20 = 520
示例：6 段 → 50 + 6×90 + 20 = 610

### 输出禁忌

- 禁止在 SVG 内部写解释性文字段落
- 禁止使用 `<style>` 块定义颜色（使用 `c-{color}` class）
- 禁止硬编码十六进制颜色值
- 禁止在任何 `<text>` 元素上省略 class 属性
- 禁止让卡片文字超出 168px 宽度（主标题≤10字，副标题≤16字）
- 禁止将两个不同段落设为同一颜色
- 垂直箭头必须三列都画，不可只画中列
- 禁止将水平箭头画为双向——感觉汇聚方向严格为 左→中→右
- 禁止在右列"调谐效应"卡片中写画面描述——右列只写观众被调谐到的情感状态
- 禁止使用叙事性标签（如"主角""高潮""结局"）——情绪蒙太奇的段落用情感状态命名，不用叙事功能命名
